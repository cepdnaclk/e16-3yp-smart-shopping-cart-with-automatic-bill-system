Desktop app

This desktop application is for the supermarket management.In our system payment are done in one check point near the cashier.

From the cart according to the cart ID final billing is sent to the desktop application the customer can pay their bill.


Technoogies used:



Electron JS:

Electron is an open-source software framework developed and maintained by GitHub. It allows for the development of desktop GUI applications using web technologies.It builds cross-platform desktop apps with JavaScript, HTML, and CSS.



Main functions of the desktop application


  • Manage transactions

  • Receipt Printing.

  • Search for product by barcode.

  • Staff accounts and permissions.

  • Products and Categories.

  • Basic Stock Management.

  • Open Tabs (Orders).

  • Customer Database.

  • Transaction History.

  • Filter Transactions by Till, Cashier or Status.
